# Ultrasonic_Horn

Conventional vehicle horns contribute significantly to noise pollution, especially in urban environments. To address this, I developed a **pollution-free ultrasonic horn system** that replaces audible alerts with high-frequency ultrasonic signals (>20 kHz), which are inaudible to humans and animals.

🔧 **What it does:**

* Uses ultrasonic transducers to transmit alert signals
* Enables proximity-based detection for nearby vehicles
* Reduces noise pollution while maintaining road safety

⚙️ **Key Engineering Aspects:**

* Embedded system design for real-time signal generation
* Ultrasonic sensing and time-of-flight based detection
* Control logic for responsive alert triggering

🏆 This project was awarded **2nd prize at a national-level hackathon**, recognized for innovation and real-world impact.

📌 **Key Insight:**
Building a quieter system is easy. Building a **reliable, detectable, and scalable alternative to horns** is the real challenge — especially when considering signal range, environmental factors, and system adoption.

This project was a step toward exploring that challenge.
